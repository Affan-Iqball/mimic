I built this game for one simple reason: **Family Game Night.**

1.  **Locked behind paywalls** (subscriptions just to get new words? No thanks).
2.  **Boring & Repetitive** (same generic "Apple/Orange" words).
3.  **Lacking Flavor** (completely missing the cultural context we joke about).

It mixes global pop culture (*Netflix/YouTube*) with the things we actually talk about (*Biryani/Pulao, Arranged Marriage/Love Marriage*). It's built to be the most fun way to determine who among us is the imposter.
